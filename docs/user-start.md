# Как запустить Clip

Эта инструкция нужна для пользователя, который скачал готовый релиз.

1. Скачать `Clip-win-x64.zip` из раздела Releases.
2. Распаковать архив в любую папку.
3. Открыть распакованную папку.
4. Запустить `Start Clip.cmd` или `Clip.exe`.

Если Windows показывает предупреждение SmartScreen, нужно выбрать дополнительные сведения и подтвердить запуск. Это обычная ситуация для учебного unsigned-приложения.

Не нужно искать файл внутри папок `bin`, `obj`, `Release` или `Debug`. В релизном архиве запускной файл лежит сверху.
